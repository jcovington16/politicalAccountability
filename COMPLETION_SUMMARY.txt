## 🏗️ Final Architecture

### Microservices
✅ **API Gateway** - REST endpoints (4 resources, 13+ endpoints)
✅ **Ingestion Service** - Data collection (4 connectors)
✅ **Processing Service** - Data enrichment (enricher pattern)
✅ **Storage Service** - Data persistence (repository pattern)

### Data Flow
```
External Sources → Ingestion → Kafka (raw-content)
→ Processing → Kafka (processed-content)
→ Storage (PostgreSQL + Elasticsearch)
→ API Gateway → Clients
```

### Technology Stack
- **Framework**: Dropwizard (REST/HTTP)
- **Language**: Kotlin + Java
- **Database**: PostgreSQL
- **Search**: Elasticsearch
- **Messaging**: Kafka
- **Storage**: MinIO
- **Build**: Gradle
